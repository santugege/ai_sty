# Dark Creative Studio Design

## Goal

Turn the project from a plain white utility page into a visual-first AI image creation studio. The first screen should make users want to create by showing image outcomes, not by presenting forms.

## Direction

Use a dark immersive creative workspace: charcoal background, graphite panels, off-white text, cyan focus accents, coral generation actions, and small gold highlights. The interface should feel closer to an image creation product than a SaaS settings page.

## Homepage

- Left rail: compact brand, mode switch, tool navigation, recent queue.
- Center: inspiration gallery with large visual tiles for product ads, portraits, restoration, lifestyle, and ecommerce banners.
- Right panel: model, platform, ratio, style chips, and prompt presets.
- Bottom composer: upload action, prompt field, ratio control, and prominent generate button.

## Tool Pages

- Keep the same backend behavior and form submission contracts.
- Replace white shells with a dark studio layout.
- Put the result canvas in the strongest visual position.
- Show default sample visuals in empty states so the page never feels blank.
- Keep controls dense and scannable.

## Constraints

- Do not change the image generation API contract.
- Do not introduce new UI dependencies.
- Keep lucide icons.
- Preserve accessibility for labels, form controls, radio groups, and buttons.
